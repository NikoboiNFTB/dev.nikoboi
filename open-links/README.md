# Bulk Open Links  

**Bulk Open Links** is a lightweight, open-source web tool that allows you to quickly open multiple URLs at once. Simply paste full links (one per line) into the text box, and the app will open all of them in new browser tabs.  

This tool is excellent to use as an add-on for Link Gopher, available for [Firefox](https://addons.mozilla.org/en-US/firefox/addon/link-gopher/) and [Chromium](https://chromewebstore.google.com/detail/link-gopher/bpjdkodgnbfalgghnbeggfbfjpcfamkf).

## ðŸ’¡ Usage  

### Two Ways to Get Started:  

1. **Visit the live site**  
   You can directly use the tool on the web by visiting [GitHub Pages](https://nikoboinftb.github.io/Bulk-Open-Links).

2. **Download the repository**  
   You can also download the source code and run it locally:
   - Clone the repository using Git:  
     `git clone https://github.com/NikoboiNFTB/Bulk-Open-Links`  
   - Or [download](https://github.com/NikoboiNFTB/Bulk-Open-Links/archive/refs/heads/main.zip) it as a ZIP file.  

   Open index.html in any browser.

### How to Continue (same for both options):  

1. Paste one or more URLs in the text area (each on a new line).  
2. Click **Open All Links** to launch them in new tabs.  
3. Click **Clear Links** to reset the input.  

**Example input:**  
https://nikoboinftb.github.io/Bulk-Open-Links  
https://nikoboinftb.github.io/Bulk-Video-Player  
https://nikoboinftb.github.io/MyLinks?
